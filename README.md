
A simple cef3 client with Qt.

![](https://github.com/joinAero/qtcefclient/blob/master/qcef3_1650.png?raw=true)

* 一个跨平台的客户端,基于Qt/QML开发
* 后端框架采用Nodejs+Express+mongodb+C++
* 与多多指教社区数据共享

Build
-----

* Qt 5.3.1
* DDui 1.0.0.0

You can download DDui here: [DDui Builds](http://heilqt.com/).

* Extract the following and copy to `DDui/`.

```
Debug/
Release/
include/
Resources/
```


* Open `DDui/DDui.pro`, then build. (Mingw4.8.x)
